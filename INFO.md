# My Sparks Project Information

## Overview
My Sparks is a mobile application built with Expo and React Native. The project is designed as a cross-platform application that can run on iOS, Android, and web platforms using a single codebase.

## Project Details
- **Name**: my-sparks
- **Version**: 1.0.0
- **Main Entry Point**: expo-router/entry

## Key Technologies & Dependencies

### Framework
- **Expo**: A framework and platform for universal React applications (version ~54.0.20)
- **React Native**: Framework for building native apps with React (version 0.81.5)
- **React**: JavaScript library for building user interfaces (version 19.1.0)

### Navigation & UI
- **React Navigation**: Complete navigation solution for React Native (bottom-tabs v7.4.0, native v7.1.8)
- **Expo Router**: File-based routing for React Native applications (version ~6.0.13)
- **@expo/vector-icons**: Icon library for Expo applications (version ^15.0.3)
- **Expo Linear Gradient**: Component for creating gradient backgrounds (version ~14.0.1)
- **React Native Gesture Handler**: Native touch and gesture system for React Native (version ~2.28.0)

### Backend & Authentication
- **Appwrite**: Backend server with REST APIs and real-time support (version ^21.3.0)
- **React Native Appwrite**: React Native SDK for Appwrite (version ^0.17.1)

### Additional Features
- **React Native Markdown Display**: Component for rendering Markdown content (version ^7.0.2)
- **React Native Worklets**: Provides access to worklets in React Native (version 0.5.1)
- **React Native URL Polyfill**: Adds URL and URLSearchParams support to React Native (version ^3.0.0)

## Scripts
- **start**: Starts the development server with `expo start`
- **android**: Runs the app on an Android emulator/device
- **ios**: Runs the app on an iOS simulator/device
- **web**: Runs the app in a web browser
- **build**: Exports the project for web deployment
- **lint**: Lints the code using Expo's ESLint configuration
- **reset-project**: Executes a custom script to reset the project

## Development Dependencies
- **TypeScript**: Type-safe JavaScript with support for modern JavaScript features (version ~5.9.2)
- **ESLint**: Tool for identifying and reporting on patterns found in ECMAScript/JavaScript code (version ^9.25.0)

## Platform Support
The application supports three platforms:
- iOS
- Android
- Web

## Project Type
This is a private project intended for cross-platform mobile and web development using the Expo ecosystem.
